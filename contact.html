Get in Touch

Send us an email at contact.unblockedarcade@gmail.com for any of the following. Use the templates below to help us respond faster.

🐛 Bug Report

Reporting a Bug

Found something broken? Include as much detail as possible so we can reproduce it. Use this format:

Subject: [Bug] <Game Name> — <short description> Game: e.g. Survive Device / Browser: e.g. Windows 11, Chrome 124 What happened: Describe exactly what went wrong. What you expected to happen: Describe what should have happened instead. Steps to reproduce: 1. Open the game 2. ... 3. Bug occurs (Optional) Screenshot or screen recording attached.

The more detail you include, the quicker we can fix it.

💡 Game Suggestion

Suggesting a Game

Have an idea for a new game? We'd love to hear it. Use this format:

Subject: [Suggestion] <Game Idea Name> Game idea: e.g. Platformer Single or multiplayer: e.g. Singleplayer Description: Explain the gameplay — how it works, what the goal is, and what makes it fun. (Optional) Any reference games or inspirations.
✉️ General

Everything Else

For anything else — questions, feedback, or just to say hi — email us at contact.unblockedarcade@gmail.com. No strict format needed, just write naturally.

We try to respond within a few days.
